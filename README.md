# Какой ИИ тебе нужен прямо сейчас?

Интерактивный инструмент для подбора ИИ-инструмента под конкретную задачу.

**3 вопроса → персональная рекомендация с инсайтом от практика.**

## Как использовать

Просто открой [страницу](https://[твой-username].github.io/[repo-name]/) и ответь на 3 вопроса.

## Как задеплоить самому

1. Создай новый репозиторий на GitHub
2. Загрузи `index.html` в корень репозитория
3. Зайди в Settings → Pages → Source → выбери `main` ветку
4. Готово — через 2 минуты сайт доступен по ссылке
